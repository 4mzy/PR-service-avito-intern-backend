# Тестовое задание на стажировку в Авито (Backend)

Backend-сервис для автоматического назначения ревьюверов для Pull Request'ов.

## Запуск (предварительно скачайте проект)

1. Запустите сервис:

```bash
make docker-up
```

Это поднимет PostgreSQL, применит миграции и запустит HTTP сервер на порту 8080.

2. Проверьте работоспособность:

```bash
curl http://localhost:8080/health
```

## Остановка

Остановите сервис:

```bash
make docker-down
```

## Дополнительные задачи

Выполнено **3 из 5** дополнительных задач:

**Endpoint статистики** - `/stats` для получения статистики по пользователям  
**Деактивация пользователей** - `/users/deactivate` с автоматическим переназначением ревьюверов  
**Конфигурация линтера** - настроен `golangci-lint` (`.golangci.yml`)

## Технологии

- Go 1.21
- PostgreSQL 15
- Docker Compose
